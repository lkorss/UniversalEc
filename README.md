# UniversalEc
Open ender chests anywhere using the command or by right-clicking in the air.

## Support Paper/Spigot/Bukkit


| Action        | Permission             | Short Description                                                             |
| ------------- | ---------------------- | ----------------------------------------------------------------------------- |
| /ec           | universalec.command.ec | When you type /ec, it opens an ender chest if you have one in your inventory. |
| 'right click' | universalec.use        | Right-clicking in mid-air opens an ender chest.                               |
| /unec reload  | universalec.admin      | Reloads the plugin configuration and localization.                            |
